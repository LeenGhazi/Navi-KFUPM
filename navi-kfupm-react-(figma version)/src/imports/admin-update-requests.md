Process Administrator Update Requests
Description:The system shall allow the Technical Admin to receive and implement update requests submitted by KFUPM Administrators to ensure that official campus information is updated accordingly.
Normal Flow:
The KFUPM Administrator submits a request to update building information, announcements, or campus data.
The system stores the request with status “Pending.”
The Technical Admin opens the Navi-KFUPM system.
The Technical Admin navigates to the “Administrator Requests” page
The system displays all submitted requests
The Technical Admin selects a request with status Pending
The Technical Admin changes the request status to In Progress
The Technical Admin navigates to the relevant management page (e.g., Edit Map or Announcements).
The Technical Admin applies the requested changes
The system updates the database
The system updates the request status to Completed

Manage Campus Buildings
Description:The system shall allow the Technical Admin to modify buildings information through the map management interface
Normal Flow:
The Technical Admin logs into the system.
The Technical Admin navigates to the  Edit Map
The system displays the interactive campus map.
The Technical Admin selects a building from the map
The system displays the building information panel.
The Technical Admin selects the section to modify (e.g., name, description,services, operating hours)
The Technical Admin edits the required information
The system updates the building information in the database

 Add Map Filters Categories
Description:The system shall allow the Technical Admin to create, update, and manage service based filters that help users search buildings by category (e.g., academic buildings, restaurants, cafes, dormitories, parking).
Normal Flow:
The Technical Admin logs into the system.
The Technical Admin navigates to the “Filter Management” page
The system displays existing service categories
The Technical Admin selects Add New Filter
The Technical Admin enters the filter name (e.g., Cafes, Parking)
The new filter category is stored in the database
