# Navi-KFUPM Front-End Prototype

This is the Phase 4 front-end prototype for **Navi-KFUPM**, adapted from the uploaded Figma-exported React code and kept as a front-end-only project for now.

## Included pages

- Landing / role selection
- Home page
- Campus map
- Bus routes
- Announcements
- Complaints
- About
- Account / profile
- KFUPM administrator pages
- Technical team dashboard pages

## Tech stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Radix UI / custom UI components

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- This version focuses on the **front-end design and flow** only.
- Backend integration, database, authentication, and live map data can be added later.
- Some files exported by Figma are kept because they are used by the running app.
